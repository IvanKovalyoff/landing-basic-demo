# Starter Landing Page – Szablon dla Małego Biznesu

## 📌 Co to jest?

To jest gotowy szablon strony typu **Landing Page**, przygotowany do szybkiego uruchomienia dla małego biznesu (np. piekarnia, salon kosmetyczny, firma budowlana, serwis samochodowy itp.).

Projekt oparty jest na technologii React + Vite + TypeScript i jest zoptymalizowany pod szybkie wdrożenie na Netlify.

---

## 🎯 Do czego służy ten projekt?

Ten starter umożliwia:

- szybkie uruchomienie profesjonalnej strony internetowej
- zbieranie zapytań od klientów (formularz kontaktowy działa przez Netlify Forms)
- prezentację oferty w przejrzysty sposób
- budowanie zaufania dzięki sekcji „O nas”, zaletom i opiniom
- zwiększenie liczby klientów poprzez obecność online

---

## 🚀 Dlaczego warto mieć własny landing page?

### 1️⃣ Więcej klientów
Strona internetowa działa 24/7 i pozwala klientom znaleźć Twoją firmę w Google.

### 2️⃣ Profesjonalny wizerunek
Nowoczesny landing zwiększa wiarygodność firmy i buduje zaufanie.

### 3️⃣ Kontrola nad treścią
Możesz samodzielnie zmieniać dane w pliku:
```
src/data/site.ts
```
Bez konieczności modyfikowania całego kodu.

### 4️⃣ Szybkość działania
Strona jest lekka, szybka i zoptymalizowana pod SEO.

### 5️⃣ Brak skomplikowanego backendu
Nie potrzebujesz serwera ani bazy danych – formularz działa bez dodatkowej konfiguracji backendowej.

---

## 🧩 Jak działa struktura projektu?

Najważniejszy plik:
```
src/data/site.ts
```
W tym miejscu znajdują się:
- dane kontaktowe
- teksty sekcji
- oferta
- CTA
- linki społecznościowe

Sekcje strony tylko renderują dane z tego pliku.

To pozwala szybko tworzyć nowe projekty poprzez zmianę jednego pliku.

## 🌐 Co zawiera projekt?

- Responsywny design (mobile / tablet / desktop)

## 🔧 Jak uruchomić lokalnie?

1. Zainstaluj zależności:
```
npm install
```

2. Uruchom projekt:
```
npm run dev
```

3. Wejdź w przeglądarce na:
```
http://localhost:5173
```

---

## 🚀 Jak wdrożyć na Netlify?

1. Wgraj projekt na GitHub.
2. W Netlify wybierz „New site from Git”.
3. Build command:
```
npm run build
```
4. Publish directory:
```
dist
```
